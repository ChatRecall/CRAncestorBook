# Sample Biography

This folder contains a sample biography created with the CRAncestorBook workflow.

The sample is included to show the kind of long-form output this project can produce from interviews, transcripts, archival material, and other family-history sources.

This is not placeholder demo text. It is based on a real family biography project that inspired the development of CRAncestorBook.

## What This Sample Shows

The sample biography demonstrates:

- how source material can be shaped into a full narrative
- how a biography can be organized into chapters
- what the overall tone and structure of the workflow output can look like
- the kind of result you can review, revise, and build on

## Included Files

This folder may include formats such as:

- EPUB
- PDF

These are provided so you can quickly review the output in a familiar reading format.

## Important Note

The **software and prompts** in this repository are open source.

The **sample biography content itself is not open source for reuse**. It is shared here only as a demonstration of the workflow and as the real-world project that motivated the program.

Please do not copy, republish, or reuse the sample biography text without permission.
